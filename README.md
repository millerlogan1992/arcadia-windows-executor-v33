## About Arcadia

Arcadia is a Roblox script executor for Windows users who want a streamlined desktop application for running Lua scripts inside Roblox. It pairs one-click Lua injection with a small, practical UI so scripts can be managed without a bulky or distracting workflow.

The project also emphasizes script hub browsing, batch execution, and a saved script queue so frequently used tools remain available across sessions. With auto-update support and a multi-language interface, Arcadia is built to stay usable for people who prefer a modern Windows executor setup.

## FAQ

**Is Arcadia limited to Windows?**  
Yes. The current release is intended for Windows desktop use.

**Do scripts continue working after Roblox updates?**  
Compatibility can shift when Roblox or the script environment changes. The auto-update engine is meant to lower manual upkeep, but script results can still vary.

**How does Arcadia differ from a simple script runner?**  
Arcadia adds a script hub, persistent queue, batch execution, debugger tools, and a lightweight desktop UI instead of only a single launch action.

**Can using a script executor increase account risk?**  
Any third-party executor and script use can involve account-related risk depending on how it is used and on platform rules. Review the relevant terms before use.

**Does Arcadia keep scripts on the device?**  
Yes. Features such as the persistent queue and local app structure indicate local storage for selected scripts and settings on the device.
